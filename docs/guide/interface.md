A tour of the Lurker window and how messages are organized.

## Buffers

- What a buffer is: channels, direct messages, and the system buffer.
- The buffer list: pinning, rearranging, and unread indicators.

## Reading and writing

- Message composition, drafts, and tab completion.
- Saved messages and user notes.

## Direct messages & presence

- Starting a DM and reading peer presence (online / away / offline).
- Sending a file or opening a direct DCC chat from the DM header or a nick's right-click menu (when DCC is enabled).

## Files & DCC

- Peer-to-peer file transfers and DCC chat, tracked in the Transfers panel.
- Browsing another user's file server (fserve) over a DCC chat, or serving your own directory. See the [self-hosting guide](/SELF_HOSTING).

## Customizing the look

- The 40+ settings and the retro terminal theme.
- Message routing (**Settings → Chat**): send notices to the server/status buffer or the buffer you're actively viewing, and auto-join / rejoin behavior.
